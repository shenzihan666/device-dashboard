# Runbooks

Operational procedures for deploying, monitoring, and troubleshooting the connection dashboard. Each runbook is a step-by-step guide intended for the operations or development team.

## Process

1. Copy the [runbook template](../templates/runbook.md) into this folder.
2. Name it with a descriptive kebab-case name (e.g., `incident-response.md`).
3. Fill in all sections.
4. Update the index table below and in [`docs/README.md`](../README.md).

## Index

| Runbook | Audience | Description |
|---------|----------|-------------|
| [Deployment](./deployment.md) | Ops / Dev | Deploy, rollback, and environment configuration |
| [Troubleshooting](./troubleshooting.md) | Ops / Dev | Common issues and resolution steps |
